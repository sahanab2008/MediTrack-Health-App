# MediTrack

MediTrack is a healthcare management web application that helps users manage medications, compare pharmacy prices, and access emergency assistance.

## Live Demo
https://app-bq6p5622eio1.appmedo.com/login

## Features

- Medication reminders
- Medicine inventory tracking
- Pharmacy price comparison
- Emergency assistance
- AI-assisted healthcare support

## About

This project was independently built using the MeDo AI development platform with assistance from Gemini and ChatGPT for ideation, debugging, and implementation.

The application demonstrates my ability to design, iterate, and build a complete web application using modern AI-assisted development tools.
